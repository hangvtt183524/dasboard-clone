# RGI - Dashboard

## Project setup
```
npm install
```
### Customize configuration
- Open src/api/request.ts
Change domain api at variable: baseURL
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies
```
npm run build
```

### Deploy Server
#### Staging
- At branch staging
```
npm run deploy-stag
```
#### Production
- At branch release
```
npm run deploy-prod
```
### Lints and fixes files
```
npm run lint
```
